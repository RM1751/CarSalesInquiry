## 🚀 Deployment Options

### Option 1: GitHub Pages (Free)
1. Push this repository to GitHub
2. Go to Settings → Pages
3. Select "Deploy from a branch" → "main" → "/ (root)"
4. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Vercel (Free)
1. Go to [vercel.com](https://vercel.com)
2. Import your GitHub repository
3. Deploy with default settings
4. Get a free `.vercel.app` domain

### Option 3: Netlify (Free)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the project folder
3. Site will be live instantly

### Option 4: Hostinger/GoDaddy (Paid)
1. Purchase a domain (e.g., `rahulmarutisiddharthnagar.in`)
2. Upload files via FTP/cPanel File Manager
3. Point domain to hosting

## 📝 Customization Guide

### 1. Add Profile Photo
```
1. Add your photo to the 'assets' folder
2. Rename it to 'profile-photo.jpg'
3. Recommended: 400x400px square photo
4. Refresh the website to see it
```
If no photo is added, a default user icon will display.

### 2. Update WhatsApp Number
Replace all instances of `919876543210` with your actual WhatsApp number (with country code, no + sign).

Search for: `919876543210` and replace with your number.

### 3. Update Personal Information
- Name: Already set to "Rahul Mishra"
- Location: Already set to "Siddharth Nagar"
- Update stats if needed (years of experience, deliveries)

### 4. Update Testimonials
Replace the customer testimonials with real ones from your clients in Siddharth Nagar.

### 5. Update Offers
Modify the offers section to reflect current month's deals.
