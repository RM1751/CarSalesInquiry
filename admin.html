How to Add Photos

  1. Go to your Cloudinary Dashboard → Media Library
  2. Click on any image you want to use
  3. Copy the URL (starts with https://res.cloudinary.com/...)
  4. Paste the URL in the input box above
  5. Click "Add" - photo will appear on the main website!
